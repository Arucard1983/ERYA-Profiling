ERYA Profiling : Handling Import Experimental Data and Images



When ERYA Profiling finishes the numerical simulation, the results are plotted on “Output” tab (this one), and a numerical table are stored on “Table” tab. A full detailed partial calculations are also stored on the “Log” table.


Export the Yield Spectrum as Images


The main window on the “Output” tab is the graphic plot of all yield spectrum plotted for each element/isotope in function to their energy, during the numerical simulation.

The user can use the pop-up menu, by right-click on the graphic output, to make some operations, like zoom, fit, and other functions provided by the software.

Tip: The user should click and drag the rectangle selection to make the majority of transformation operations.

Once make the necessary operations, just click the “Save to Image” button, and select one of four common image file formats (JPG, PNG, TIFF and BMP), and save the file with a given name.

If necessary, repeat the same operations to make another image with another custom selection.


Adding Experimental Data


It’s possible to add experimental data from a ASCII or an Excel Xlsx file, as long they have two numerical columns of data. In case of a successful data importation, the program will add another plot, and a name derived from “ExpLayer<n>” (n is an integer) directly on the graph plot output, without deleting the previous ones. Notice that running a new numerical simulation will delete all output contents, so make sure to add after the numerical simulation.
