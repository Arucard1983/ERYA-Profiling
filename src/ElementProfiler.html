ERYA - Profiling : Element Profile Editor


On “Elements” tab from the main program screen, the user can define an array of several elements loaded from the Default Database. This screen also makes possible to override any of the five Elements physical parameters, without editing the original Database data. This ease some simulations that had samples with a particular mixture of isotopes that diverge from the standard values stored on the Database or other sources.

All user changes on this page can be stored on an XML file with epcs (ERYA Profiling Compound Sample) format, which is the recommended setting, or an Excel file.

Any custom profiles requires forehand that the Elements names exists on the same Database, before replace the custom values (but it will not overwrite the original values, since it was stored on different memory locations). Otherwise, an error will be issued to warn the user to rectify before resumes.


Adding and Removing Elements


To change the contents on this editor, use the “Number of Elements” switch to increase/decrease the number of Elements. Notice that deleting some elements, will not reset the previous selection. And adding new ones, won’t reset the previous defined Elements.


Either way, all changes on “Elements” tab will reset the Layers Editor spreadsheet contents, then it is recommended to define all necessary elements first, before continue to the next tab.
