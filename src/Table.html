ERYA Profiling : Handling Import Experimental Data and Tables



When ERYA Profiling finishes the numerical simulation, the yield spectrum is plotted as a graph on “Output” tab, and a numerical table of the same results are stored on “Table” tab(this one). A full detailed partial calculations are also stored on the “Log” table.


View and Export the Table Data


The yield spectrum data are represented on a spreadsheet table, where the first column is the Energy (actually the initial energy for each point), and the other columns represents each Element/Isotope yields.

The contents of this table can be stored to an ASCII or Excel Xlsx file, using the “Save Table Data” or “Export Results to File” buttons.


Adding Experimental Data


It’s possible to add experimental data from a ASCII or an Excel Xlsx file, as long they have two numerical columns of data. In case of a successful data importation, the program will add another plot on the “Output” tab, and a new column, with a name derived from “ExpLayer<n>” (n is an integer), without deleting the previous data. Notice that running a new numerical simulation will delete all output contents, so make sure to add after the numerical simulation.
